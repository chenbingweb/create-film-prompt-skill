# Trae Agent Skills 使用说明文档

本目录包含了两个专门为图像和视频生成设计的 Agent 技能。

## 1. 图片风格提示词生成器 (image-style-prompt-generator)
此技能用于为 ComfyUI 和 ChatGPT (DALL-E 3) 生成高质量、具有特定艺术风格的图像提示词。

### 核心功能
- **海量风格库**：内置 126 种艺术风格（如：赛博朋克、敦煌艺术、皮克斯、宫崎骏、国风等）。
- **多平台适配**：同时生成适用于 ComfyUI (关键词权重格式) 和 ChatGPT (自然语言格式) 的提示词。

### 使用方法
直接提供主题和风格，例如：
> "帮我生成一个关于'未来城市'的提示词，使用'赛博朋克'风格。"
> "用'宫崎骏风格'画一只猫，生成提示词。"

---

## 2. 爆款短视频脚本生成器 - 商品版 (viral-video-script-generator)
此技能专为商品营销设计，将商品图与经典虚拟角色结合，生成 6 秒爆款短视频脚本及配套提示词。

### 核心功能
- **商品深度融合**：自动解析上传的商品图视觉特征（颜色、材质、包装），并将其植入剧本。
- **经典角色互动**：使用孙悟空、唐僧、林黛玉、伏地魔等经典形象进行趣味带货对话。
- **全链路提示词输出**：
  - **分镜剧本**：带动作和台词的 6 秒内短视频脚本。
  - **画面提示词 (Image Prompt)**：直接用于 AI 绘画，确保角色与商品视觉统一。
  - **视频提示词 (Video Prompt)**：包含运镜和动态指令，用于 AI 视频生成。

### 使用方法
提供：**风格 + 商品图 + 卖点/话题**，例如：
> "风格：皮克斯。商品图：红色运动鞋。话题：孙悟空想穿这双鞋去取经。"
> "风格：赛博朋克。商品图：绿色能量饮料。话题：哪吒熬夜打游戏的续命水。"

---

## 如何安装/更新技能
这些技能定义在 `.trae/skills/` 目录下的 `SKILL.md` 文件中。Trae Agent 会自动读取并根据您的指令调用这些能力。如果您需要添加新风格或修改流程，直接修改对应的 `SKILL.md` 文件即可。
